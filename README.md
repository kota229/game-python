# 🤖 CodeQuest — 子ども向けプログラミング学習ゲーム

ロボット「コード」にめいれいを出して、マス目マップのゴールへ導く子ども向けの
プログラミング学習ゲームです。ビジュアルブロックから本物のPythonまで、
**4段階のカリキュラム**で「Pythonの基本文法を読み書きできる」ことを目指します。

- 対象: 小学3年生〜中学生 / タブレット横画面のタッチ操作前提
- **オフライン完結**（初回ロード後は外部通信なし）・**ビルド不要の静的構成**
- 進捗はブラウザの localStorage に自動保存

> 現在は **フェーズ1（プロトタイプ）** の段階です。段階1（ビジュアルブロック）の
> 5ステージ（順次実行→くりかえし）が遊べます。段階2〜4（ブロック⇔Python／穴埋め／
> 自由記述）と全33ステージはフェーズ2で実装予定です（`docs/design.md` 参照）。

---

## あそびかた（操作説明）
1. **はじめる** を押してステージをえらぶ（クリアすると次のステージが解放されます）。
2. プレイ画面の右上「つかえるブロック」から、下の「じっこうするめいれい」へ
   ブロックを **タップで追加** または **ドラッグ** して並べます。
   - `くりかえす` ブロックの中には、ブロックを **ドラッグして入れ子** にできます（回数は −／＋ で変更）。
   - ブロックの `✕` で削除、`⠿` をドラッグで並べ替え。
3. **▶ じっこう** を押すと、ロボットが命令を1つずつ実行してアニメーションで動きます。
4. ゴール（🚩）に着く／ジェム（💎）を全部集めるとクリア。**スター1〜3個**で評価されます。
   - 少ないブロックで解くほど★が多い（`くりかえし` を使うと短く書けます）。
5. こまったら **💡ヒント**（3段階：やさしい→答えに近い→模範解答）。失敗してもペナルティなし。

## 起動方法

### A) ブラウザで遊ぶ（GitHub Pages / 主要納品経路）
GitHub Pages に公開すると、URL をタブレットのブラウザで開くだけで遊べます。
公開URLは初回セットアップ後に `DECISIONS.md` に記録されます
（形式: `https://<ユーザー名>.github.io/game-python/`）。

### B) ローカルで遊ぶ
ES モジュールを使うため `file://` 直開きではなく簡易サーバ経由で開きます（依存ゼロ）。
```bash
npm run serve        # → http://localhost:8080 を開く
```
Node が無い場合は任意の静的サーバ（例: `python -m http.server`）でも可。

### Androidタブレットでの動作
- Chrome 等のブラウザで上記URLを開くだけ。**横向き**推奨。
- ブラウザメニューの「ホーム画面に追加」でアプリのように起動できます（PWA対応）。

## APK化（Capacitor）※手順書
ブラウザ版がそのまま動くため、必要に応じて Capacitor で APK 化できます
（本リポジトリは静的構成なので Web アセットをそのまま `webDir` に指定できます）。
```bash
npm install -D @capacitor/core @capacitor/cli @capacitor/android
npx cap init CodeQuest com.example.codequest --web-dir .
npx cap add android
npx cap sync
npx cap open android   # Android Studio でビルド → APK 生成
```
> 実際の APK ビルドには Android Studio / JDK が必要です。フェーズ2（M8）で構成を整備します。

## 技術構成（概要）
- **フロント**: HTML5 + プレーンJavaScript（ES モジュール、ビルド不要）
- **実行エンジン**: 自作のトレースベース・インタプリタ（DOM非依存・テスト可能）
- **グラフィック**: Canvas による自作描画 / **効果音**: Web Audio API によるプログラム生成
  （画像・音の外部アセット流用は一切なし）
- **Python実行（段階4・予定）**: Pyodide 第一候補 / Skulpt を軽量代替として検証（`DECISIONS.md`）
